CaRMS Match Rates

From the CaRMS website: "The Canadian Resident Matching Service (CaRMS) is national, independent, not-for-profit, fee-for-service organization that provides a fair, objective and transparent application and matching service for medical training throughout Canada."


In the .R files, I analyze the match rates for each year and across time for the following medical schools: Queen's University, University of Toronto, McMaster University, University of Alberta, University of Calgary, University of British Columbia, University of Ottawa. You can easily add additional schools to analyze by modifying the relevant sections in the code. Whereas big_analysis.R uses the data I've compiled in this table row, carms_match_rate.R pulls directly from the CaRMS website. Run the code using the R programming language to view the figures.
